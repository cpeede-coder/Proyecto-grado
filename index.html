🤖 Corrección automática con IA (opcional)

Si guardas aquí una API key gratuita de Google Gemini, al terminar cada examen aparecerá el botón "🤖 Corregir con IA": evalúa tus respuestas contra la pauta oficial (entiende paráfrasis y sinónimos), marca los criterios por ti y te da feedback por pregunta. Siempre puedes ajustar el resultado a mano.

Cómo crear tu API key — gratis, 1 minuto, sin tarjeta:
  1. Entra a aistudio.google.com/apikey e inicia sesión con tu cuenta de Google.
  2. Presiona "Create API key" (o "Crear clave de API"). Si te pide elegir un proyecto, acepta el que te sugiere.
  3. Copia la key generada (empieza con AQ. o AIza).
  4. Pégala aquí abajo y presiona Guardar key. ¡Listo!

🔒 Tu key se guarda solo en este navegador — no se sube a ningún servidor y solo se usa para llamar a Google cuando corriges. Cada persona debe crear la suya (no compartas la tuya): es gratis y así cada uno tiene su propia cuota diaria, que alcanza de sobra para varios exámenes al día.
